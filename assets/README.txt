Chess Pieces Assets
===================

Source:
Chess pieces images downloaded from Vecteezy:
https://www.vecteezy.com/vector-art/49089132-chess-pieces-king-queen-bishop-knight-rook-and-pawn-and-bishop-success-strategy

Processing:
The original vector images were split into individual transparent PNG files
for use in the Pygame chess GUI.

Structure:
- chess_pieces/           - Processed transparent PNG files (used by game)
  - originals/           - Original source files from Vecteezy (backup)
  - *.png                - 12 chess piece PNGs (6 white + 6 black)

- splashscreen/          - Custom boot/shutdown splash screen for Raspberry Pi
  - splash_1024x614.png  - Boot screen image
  - README.md            - Installation instructions

- appicon/               - Application icon
  - appicon_original_by_chatgpt.png  - App icon (generated by ChatGPT)

Files are located in: chess_pieces/

Expected files:
- White pieces: white_king.png, white_queen.png, white_rook.png, 
                white_bishop.png, white_knight.png, white_pawn.png
- Black pieces: black_king.png, black_queen.png, black_rook.png,
                black_bishop.png, black_knight.png, black_pawn.png
